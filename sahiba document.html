What is HTML

HTML stands for Hyper Text Markup Language.It is used for creating web page.HTML describe the structure and content of a Web Page.Such as "this is a heading","this is a paragraph","this is a link",etc.

  • The < !DOCTYPE html > declaration defines that this document is an HTML5 document.
  • The < html > element is the root element of an HTML page.
  • The< head > element contain meta information about the HTML page.
  • The < title > element specifies a title for the HTML page.
  • The < body > element define the document body and is a container for all the visible contents, such as heading, paragraphs, images, etc.
  • The < h1 > element defines a large heading.
  • The < p > element defines a paragraph.

CSS

CSS stand for Cascading style sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. The external stylesheets are stored in CSS files. CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.
